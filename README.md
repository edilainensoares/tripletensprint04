# tripletensprint04
Projeto de testes de API no Urban.Grocers, validando funcionalidades de produtos e entrega

No projeto criado pela Tripleten, teve como objetivo testar a API. Os desenvolvedores adicionaram um novo recurso API do Urban.Grocers, e uma nova versáo da API foi nos passada para testar.

Nesse projeto realizamos: 
* Análise de requisitos e criação de checklist para cenários de teste. 
* Execução dos testes via Postman e reporte de bugs via JIRA.

Projeto revisado e aprovado pelos instrutores.
