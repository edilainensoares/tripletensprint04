# tripletensprint04
Projeto de testes de API no Urban.Grocers, validando funcionalidades de produtos e entrega

No projeto criado pela Tripleten, tem como objetivo realizamos teste em um aplicativo de Android que permite que os clientes peçam comida de diferentes restaurantes da cidade e, depois, coletem os pedidos em um dos pontos de coleta. 

* Realizei análise de requisitos e criação de checklist para cenários de teste. 
* Foi executado os testes via Postman e reporte de bugs via JIRA.

Projeto revisado e aprovado pelos instrutores.
